💡 CodeBack 核心功能

CodeBack 不僅僅是一個貼程式碼的地方，它是一個擁有完整生態系的創作者平台。

1. 一鍵發布 (One-Click Publish)

貼上你的 HTML/CSS/JS 代碼，瞬間獲得一個可預覽的短網址。支援由 Tailwind CSS、Canvas 甚至 Three.js 驅動的單頁應用。

2. 安全創作模式 (Safe Building)

我們深知Vibe Coding時代將會出現非常大量的"Vibe Coder"，所以我們為託管的網站設下了一些安全限制：

外部連結攔截：防止惡意跳轉，保護瀏覽者安全。

沙盒環境：確保程式碼在安全的環境下執行。

不提供任何的後端與資料庫的幻想：避免API key裸奔。

3. 獨特的「續命」機制 (Time Capsule)

為了保持資料庫的健康與活躍度，每個作品都有「生命週期」。

免費加時器：每月登入即可領取，為你喜歡的作品延長壽命。

時光膠囊：重要作品可以透過膠囊長久保存。

4. 創作者生態

完整後台：管理你的所有作品。

程式碼主權：不綁架任何一條程式碼，想要離開或重新發布一個新網頁隨時都可以。

GitHub 圖床整合：如上所述，我們內建了解決圖片託管的方案。

🛠️ 技術堆疊 (Tech Stack)

這個平台本身是使用現代化的 Web 技術構建的：

Framework: Next.js 15 (App Router)

Language: TypeScript

Styling: Tailwind CSS

Database & Auth: Supabase

UI Components: Lucide React

Payments: NewebPay (藍新金流)
